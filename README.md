## What this is
CITL Library Assistant is a Windows 11–facing offline teaching tool that combines:
- **Local LLM chat** through **Ollama**
- **RAG (retrieval)** over instructor-provided .txt documents
- **Classroom audio capture** + **offline transcription** (record → transcribe → save)
### Why it matters in classrooms
- Works **offline / in-house** for privacy and reliability
- Enables instructors to drop in course readings and get **instant local Q&A**
- Supports **lecture capture workflows**: recording, storage, transcription, and searchable notes
- Provides a practical bridge from “AI curiosity” to **repeatable teaching workflows**
