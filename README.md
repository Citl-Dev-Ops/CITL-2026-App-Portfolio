## What this is
CITL Factbook v2.0 is a cross-platform offline teaching utility for Windows and Ubuntu.

It combines:
- Local LLM chat through Ollama
- Reference Q&A over Factbook and other local reference corpora
- Classroom audio capture and offline transcription
- Translation and study-support workflows
- USB and external-drive sync for CITL-managed copies

## Why it matters
- Works offline / in-house for privacy and reliability
- Gives staff one shared content set across Windows and Ubuntu
- Supports repeatable classroom and support workflows instead of one-off demos
- Makes updates easier to distribute through the CITL sync utility

## Recommended use
### Windows
- First-time setup: `INSTALL_WINDOWS.cmd`
- Daily launch: `RUN_FACTBOOK_WINDOWS.cmd`
- USB portable setup: `USB-INSTALL-CITL.cmd`
- USB portable run: `USB-RUN-CITL.cmd`
- Sync utility: `RUN_APP_SYNC_WINDOWS.cmd`

### Ubuntu
- First-time setup: `bash ./INSTALL_UBUNTU.sh`
- Daily launch: `bash ~/CITL-Factbook/RUN_FACTBOOK.sh`
- Sync utility: `bash ./RUN_APP_SYNC_UBUNTU.sh`
